# 还元堂养生中心空间规划方案 v1

## 项目目标
从零开始规划 240㎡养生空间，基调为新中式禅意风格，营造自然、温暖、疗愈的体验。

## 初步功能分区建议
> 以下为初步规划，需结合平面图尺寸和现场扫描复核。

| 功能区 | 建议面积占比 | 功能 | 配置建议 |
|---|---:|---|---|
| 玄关/接待区 | 约 8%-12% | 迎宾、登记、展示品牌气质 | 接待台、等候座、香氛/陈列 |
| 会客区/茶区 | 约 10%-15% | 洽谈、茶饮、放松等待 | 茶桌、单椅、边柜、柔光照明 |
| 禅堂/冥想室 | 约 10%-15% | 冥想、静修、小型课程 | 地台、蒲团、收纳、低照度灯光 |
| 普通治疗室 | 约 30%-38% | 3-4 间常规理疗 | 理疗床、矮柜、挂衣区、洗手点待确认 |
| VIP 治疗室 | 约 15%-22% | 1-2 间高私密服务 | 独立理疗床、茶位、储物、独立氛围灯 |
| 办公室 | 约 5%-8% | 管理、资料、员工办公 | 办公桌、文件柜、弱电点位 |
| 储物/设备间 | 约 5%-8% | 物料、设备、清洁用品 | 货架、设备位、通风需求待确认 |

## 初步动线原则
1. 顾客入口后先到接待区，再进入会客茶区或治疗区。
2. 治疗区应形成安静内区，减少与员工/储物动线交叉。
3. VIP 区宜布置在更安静、更深处的位置。
4. 储物/设备间靠近员工动线，避免穿越顾客等候区。
5. 禅堂/冥想室应避开入口噪声和设备噪声。

## 墙体调整与开门建议
- 需先确认承重墙、梁柱和消防疏散要求。
- 普通治疗室建议避免门对门直视，提升私密性。
- VIP 房门建议减少与公共等候区直接相对。
- 走廊不宜形成过长压迫感，可通过端景、灯带和材质分段缓解。
