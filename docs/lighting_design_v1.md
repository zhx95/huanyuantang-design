# 还元堂养生中心灯光设计方案 v1

## 设计目标
以间接照明、墙洗灯、磁吸轨道灯、线型灯带为主，营造新中式禅意、自然、温暖、疗愈的氛围。

## 基础参数
| 项目 | 建议 |
|---|---|
| 色温 | 2700K-3000K |
| 显色指数 | CRI >= 90 |
| 光感 | 柔和、低眩光、分层控制 |
| 预算 | 灯具采购成本单项控制在 500 元以内 |

## 各区域灯光建议
| 区域 | 灯具类型 | 功率建议 | 色温 | 控制逻辑 |
|---|---|---:|---|---|
| 玄关/接待区 | 磁吸轨道射灯、接待台灯带 | 待尺寸确认 | 3000K | 接待主灯/背景灯分控 |
| 会客/茶区 | 线型灯带、低眩光射灯、局部吊灯 | 待尺寸确认 | 2700K-3000K | 茶桌重点光/氛围光分控 |
| 禅堂/冥想室 | 间接灯带、墙洗灯 | 待尺寸确认 | 2700K | 冥想模式/清洁模式分控 |
| 普通治疗室 | 柔性灯带、可调角射灯、床头辅助光 | 待尺寸确认 | 2700K-3000K | 治疗/清洁/夜间弱光分控 |
| VIP 治疗室 | 灯带、墙洗灯、重点射灯 | 待尺寸确认 | 2700K | 接待/治疗/休息三路控制 |
| 办公室 | 线型基础照明、桌面辅助光 | 待尺寸确认 | 3000K | 办公主灯独立控制 |
| 储物/设备间 | 基础吸顶灯或线型灯 | 待尺寸确认 | 3000K | 单路开关 |

## 简易灯位示意
```text
入口/接待：  磁吸轨道灯 + 接待台下灯带
会客茶区：  茶桌重点光 + 墙面洗墙光 + 柜体灯带
禅堂：      周边暗藏灯带 + 端景墙洗灯
治疗室：    床两侧柔光 + 顶部间接光 + 清洁照明
VIP：       床区柔光 + 茶位重点光 + 背景灯带
办公/储物： 基础照明为主
```

## 成本控制建议
- 优先选择国产磁吸轨道系统、线型灯带和低眩光射灯。
- 单个灯具或单套灯带材料成本控制在 500 元以内。
- 暂不指定最终型号，需结合数量、长度、安装方式和采购渠道确认。

## 暂不确定项
- 灯具数量、准确功率和回路数量需等待平面尺寸、层高、吊顶条件、电源点位后确认。
- 不假定梁位、吊顶高度和原始线路。
