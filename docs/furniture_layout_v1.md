# 还元堂养生中心家具布置方案 v1

## 当前状态
本文件为家具布置深化的准备稿。由于关键尺寸、墙体可改性、门窗位置、电源点位尚未确认，暂不输出最终家具定位图。

## 待确认资料
- 平面图中各房间净尺寸
- 门窗位置、尺寸、开启方向
- 墙体是否可拆改
- 插座、开关、给排水、空调、新风位置
- 理疗床规格与数量
- 是否需要双人治疗室
- VIP 房是否需要独立茶位、衣柜或洗手台

## 初步家具配置建议
| 区域 | 家具/设备 | 布置建议 |
|---|---|---|
| 玄关/接待区 | 接待台、等候椅、展示柜 | 接待台正对入口或侧迎，避免堵塞动线 |
| 会客/茶区 | 茶桌、单椅、边柜 | 靠近接待但保持安静，形成停留区 |
| 禅堂/冥想室 | 地台、蒲团、矮柜 | 留出完整中心活动面，收纳靠边 |
| 普通治疗室 | 理疗床、边柜、挂衣钩 | 床两侧需预留操作空间，具体尺寸待确认 |
| VIP 治疗室 | 理疗床、休闲椅、茶几、储物柜 | 强化私密感和停留体验 |
| 办公室 | 办公桌、文件柜、打印位 | 靠近员工动线，减少进入顾客核心区 |
| 储物/设备间 | 货架、清洁柜、设备位 | 分区存放耗材、清洁用品和设备 |

## 材料与色调建议
- 木饰面：温润浅胡桃木或原木色，避免过深压抑。
- 墙面：米白、浅灰、暖砂色，局部可用肌理涂料。
- 石材：哑光浅色石材或仿石材，用于接待台/茶区重点面。
- 布艺：亚麻、棉麻、低饱和暖色系。
- 金属：少量深铜色或哑黑点缀。
